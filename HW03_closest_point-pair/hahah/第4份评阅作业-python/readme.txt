在命令行中输入“python ”+要运行的python文件路径运行该py文件

首先会出现一行“请输入一个数字，代表了点的数量，该数量大于2：”文字，在此处输入一个大于二的正整数，规定生成多少个点。

然后会出现一行“请输入四个数字，分别代表x坐标的上、下限，y坐标的上、下限（用空格分隔）：”，此处输入四个数字，分别代表所有点的x坐标的上、下限，y坐标的上、下限

然会回车

最后会得到三行数据，第一行数据为暴力算法得到的结果，第二行代表朴素的分治法的得到的结果，第三行代表改进后的分治法得到的结果。

每行的格式都为“running time is:  运行所耗时间, min-distance: 距离最小值 , 第一个点的坐标值, 第二个点的坐标值 ”
